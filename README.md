# Email Client Web App

Single-page web app that allows users to send, receive, archive, and reply to emails in a Gmail-like interface.

---

## Video Overview
[Watch the project overview](https://youtu.be/iJK6HVdRG5E)
---

## Features
- Navigate between Inbox, Sent, and Archive mailboxes
- Send emails with recipients, subject, and message body
- View full email content with sender, recipients, timestamp, and body
- Mark emails as read/unread
- Archive and unarchive emails without reloading the page
- Reply to emails with pre-filled recipients, subject, and quoted message

---

## Technologies
JavaScript, HTML, CSS, Bootstrap, RESTful APIs

---

## Installation
1. Clone the repository:
   git clone https://github.com/saraccmululo/email-client-app.git
   cd email-client-app
2. Open index.html in your browser (or serve via a local web server)

3. Ensure the backend server is running and accessible to handle API requests

Usage
Use the navigation to switch between mailboxes
Click “Compose” to send a new email
Click on any email to view its full content

Archive/unarchive emails directly from the email view

Use the “Reply” button to reply to an email with pre-filled fields
